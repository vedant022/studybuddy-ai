StudyBuddy AI Planner is an AI-powered exam preparation platform that helps students generate personalized study schedules, analyze study materials, and track their progress efficiently.

The system integrates artificial intelligence with structured planning and productivity tools to provide a complete academic assistance solution.

🚀 Features
AI-Powered Study Plan Generation
Creates personalized day-wise schedules based on subjects, exam dates, and priorities
Topic Extraction from Study Materials
Analyzes uploaded PDFs and PPTs to identify important topics automatically
Adaptive Study Planning
Updates schedules based on user progress and incomplete tasks
Daily Task Management
Provides structured checklists to ensure consistent preparation
Flashcards & Practice Questions
Generates revision material for better learning
Progress Tracking Dashboard
Visualizes completion status and study performance
Pomodoro Timer
Enhances focus with structured study sessions
🧠 How It Works
Users input subjects, exam dates, and study preferences
Upload study materials (PDFs, PPTs, notes)
AI analyzes content and extracts key topics
System generates a personalized study plan
Users follow daily tasks and track progress
🛠️ Tech Stack

Frontend: React, TypeScript, Tailwind CSS
Backend: Supabase (Auth, PostgreSQL Database, Storage)
AI Integration: Google Gemini API
Tools: Recharts, jsPDF, Vite
